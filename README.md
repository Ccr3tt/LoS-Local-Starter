# Dockerized Lord of SQL Injection Local Starter

본 프로젝트는 **Lord of SQL Injection(LoS)** 문제를 로컬 환경에서 학습할 수 있도록 Docker 기반으로 구성한 실습용 환경입니다.

모든 원본 문제의 저작권은 **Rubiya**님께 있습니다.
상업적으로 사용하고자 하는 경우, 반드시 Rubiya님께 직접 연락하여 허가를 받으시기 바랍니다.

## Docker 실행 방법

```bash
docker compose up -d --build
```

접속 주소:

```text
http://127.0.0.1:8080
```

컨테이너 종료:

```bash
docker compose down
```

풀이 기록을 유지하려면 `data/solved.json` 파일을 삭제하지 마세요.

# Dockerized Lord of SQL Injection Local Starter

This project is a Docker-based local practice environment for **Lord of SQL Injection(LoS)**.

All copyrights of the original challenges belong to **Rubiya**.
If you wish to use this project commercially, please contact Rubiya directly and obtain permission.

## How to Run with Docker

```bash
docker compose up -d --build
```

Access URL:

```text
http://127.0.0.1:8080
```

Stop the containers:

```bash
docker compose down
```

To keep your solved progress, do not delete the `data/solved.json` file.
